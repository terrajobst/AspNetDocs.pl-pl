# <a name="jquery-cycle-299"></a>jQuery Cycle 2.99

Następujące pliki cyklu jQuery są hostowane w tej sieci CDN:

- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js) &mdash; pełną wersję wtyczki cyklu jQuery. Obejmuje więcej niż dwa dziesiąte efekty specjalne i wszystkie opcje.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js) &mdash; wersji zminimalizowanego pliku powyżej. Użyj tej wersji dla aplikacji produkcyjnych, aby zwiększyć wydajność witryny sieci Web.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js)&mdash; ta wersja wtyczki programu jQuery Cycle jest mniejsza i zawiera tylko efekt przejścia "zanik".
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js) &mdash; wersji zminimalizowanego pliku powyżej. Użyj tej wersji dla aplikacji produkcyjnych, aby zwiększyć wydajność witryny sieci Web.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js) &mdash; nawet mniejszych wersji wtyczki cyklu jQuery, która zawiera tylko przejście "Zanikanie" bez wszystkich opcji.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js) &mdash; wersji zminimalizowanego pliku powyżej. Użyj tej wersji dla aplikacji produkcyjnych, aby zwiększyć wydajność witryny sieci Web.
